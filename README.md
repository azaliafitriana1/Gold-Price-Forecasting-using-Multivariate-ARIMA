# Gold-Price-Forecasting-using-Multivariate-ARIMA
Developed a multivariate time-series model (VECM) that used key macroeconomic indicators as predictors for the price of gold. Built and validated time series forecasting models, achieving a high accuracy of 2.3% MAPE. Conducted statistical diagnostics to ensure model robustness.

Dataset:
X1: Inflation (%) 
X2: Policy Rate (%)
X3: Interest Rate (%)
X4: USD to Rupiah (Rupiah)
X5: World Crude Oil per Barrel (Rupiah)
Xt: Indonesia Gold Price per Gram (Rupiah)

Source: Investing.com, Bank Indonesia
